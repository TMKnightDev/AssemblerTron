# AssemblerTron
***
An assembly encoder/decoder program written in C.

## To run
- Ensure GCC is installed on your machine, either via WSL or MinGW.
- Run the command, `gcc ADD.c ADDI.c AND.c ANDI.c BEQ.c BNE.c DIV.c LUI.c LW.c MFHI.c MFLO.c MIPS_Instruction.c MIPS_Interpreter.c MULT.c OR.c ORI.c SLT.c SLTI.c SUB.c SW.c -o MIPS_Interpreter.exe`
- Run the generated file, `MIPS_Interpreter.exe`
- Step through the command line sequence for `Assembly to Machine Code` or `Machine Code to Assembly`.
